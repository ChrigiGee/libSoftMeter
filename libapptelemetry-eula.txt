Introduction.
----------------
This software program uses the libAppTelemetry component that allows the developer of the 
software to collect installation and usage statistics.


In brief.
----------------
- No personal information is contained in the data.
- The data are not sold, nor used/given to 3rd parties.
- The data are used in aggregated form (e.g. number of installations per month).
- The platform of Google Analytics is used for the data collection and reporting.


Kind of data.
----------------
The data collected are completely anonymous and consist of parameters like the operating 
system, the version of the software, the screen resolution, the country, any errors 
occurred during the running of the program, etc.

The same type of data are commonly collected every time we visit anonymously a website. 
This is mentioned to allow you to establish an analogy between the data collected by this 
program and the commonly used data collection that is a norm on the internet.


Purpose of data collection.
-------------
The data are collected for the sole purpose of monitoring the good running of the program 
and identifying areas for improvement, discover incompatibilities with specific operating 
systems, estimate the size of the user base, plan and respond to increases in demand, etc.


Storage of the data.
-------------
Data are trasnfered directly from the program to the secure platform of Google Analytics. 
Google Analytics is used for the reporting on the data.
